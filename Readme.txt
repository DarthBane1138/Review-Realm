Paleta de colores:

Colores principales

- Azul Pastel Claro: RGB(173, 216, 230) - Un tono suave y relajante, ideal para fondos o áreas grandes.
- Azul Pastel Medio: RGB(135, 206, 235) - Un poco más vibrante, perfecto para destacar secciones o como color secundario.
- Azul Pastel Oscuro: RGB(100, 149, 237) - Un azul más profundo que puede servir para enfatizar elementos importantes como botones o enlaces.
- Gris Oscuro: RGB(105, 105, 105) - Un gris neutro y versátil, excelente para texto y elementos de interfaz.
- Gris Casi Negro: RGB(50, 50, 50) - Un gris muy oscuro que proporciona un excelente contraste, especialmente útil para texto sobre fondos claros.

Colores Adicionales

- Celeste Claro 1: RGB(135, 206, 250) - Un tono suave y luminoso que complementa muy bien los azules pasteles1.
- Celeste Claro 2: RGB(173, 216, 230) - Un poco más suave que el anterior, ideal para crear un contraste sutil con los grises oscuros.
